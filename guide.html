✈️ Panduan Penggunaan Expense Dinas

Tutorial lengkap menggunakan aplikasi pencatat pengeluaran perjalanan dinas

🚀 Mulai Cepat (5 Menit)

  1. Buka aplikasi → Klik tombol + Trip di atas
  2. Isi informasi trip
    • 📝 Nama perjalanan (contoh: "Dinas Jakarta Juni 2025")
    • 📍 Tujuan (contoh: "Jakarta")
    • 👤 Nama karyawan utama
    • 👥 Nama staff & manager - klik "+ Tambah Staff" dan "+ Tambah Manager" untuk menambah satu per satu
    • 🤝 Customer (opsional) - untuk pihak ketiga yang ikut perjalanan
    • 💰 Budget per kategori (Transport, Food, Tujuan, Overhead, Entertainment)
  3. Pilih trip di tab "Tambah" → Member checkbox otomatis muncul
  4. Tambah pengeluaran
    • Isi keterangan dan jumlah
    • 📋 Pilih tipe nota: Onsite / Grab / Gojek
    • ☑️ Centang member yang menggunakan (atau "Semua")
    • 📷 Ambil/upload foto bukti
  5. Lihat ringkasan di tab "Ringkasan"
  6. Export PDF di tab "Export"

✨ Fitur Utama

📷 Foto Bukti

Tangkap bukti pengeluaran dengan foto. Otomatis ter-record tanggal, jam, dan lokasi.

💰 Tracking Budget

Monitor pengeluaran real-time dengan progress bar dan cap category yang dinamis.

📊 Laporan Rinci

Lihat breakdown pengeluaran per kategori dan ringkasan per trip.

📄 Export PDF

Download laporan lengkap dengan semua bukti foto dalam satu file PDF.

📱 Offline Mode

Semua data tersimpan lokal di device. Tidak butuh internet untuk input data.

🔐 Data Aman

Tidak ada data yang dikirim ke server. 100% private dan aman.

📖 Panduan Detail

1. Membuat Trip Baru

  1. Klik tombol + Trip di bagian atas
  2. Isi form dengan data berikut:
    Nama Perjalanan * (wajib)
    Contoh: "Dinas Jakarta Juni 2025" atau "Kunjungan Klien Bandung"

    Tujuan (opsional)
    Contoh: "Jakarta", "Bandung"

    Nama Karyawan Utama (opsional)
    Nama yang melakukan perjalanan

    👥 Anggota Perjalanan (Baru v2.7!)
    Masukkan nama staff dan manager satu per satu:
    • Klik "+ Tambah Staff" → Isi nama staff
    • Klik "+ Tambah Manager" → Isi nama manager
    Contoh: Fajar, Reza, Budi, dll.

    🤝 Customer (opsional - v2.7)
    Nama customer/klien yang ikut perjalanan. Akan ditampilkan di laporan.

    💰 Budget Per Kategori * (wajib)
    Alokasikan budget untuk: Transport, Food, Tujuan, Overhead, Entertainment
    Contoh: Transport 500.000, Food 1.500.000, dll.
  3. Klik Buat Trip

2. Menambah Pengeluaran (dengan Member Tracking v2.7)

  1. Pilih trip di dropdown "Pilih Perjalanan" → Member checkbox otomatis muncul ✓
  2. Isi informasi pengeluaran:
    Keterangan *
    Contoh: "Taksi ke bandara", "Makan siang", "Hotel"

    Jumlah (Rp) *
    Nominal pengeluaran

    Tanggal *
    Tanggal pengeluaran (default: hari ini)

    Kategori
    Pilih kategori pengeluaran (Transport, Food, Tujuan, Overhead, Entertainment)

    📋 Tipe Nota (Baru v2.7!)
    Pilih: 🏢 Onsite / 🚕 Grab / 🏍️ Gojek

    ☑️ Member (Baru v2.7!)
    • Klik "Semua" untuk select semua member sekaligus
    • Atau centang individual member yang menggunakan nota
    • Minimal pilih 1 member
  3. Upload foto bukti (opsional tapi direkomendasikan)
    💡 Tip: Saat upload foto, aplikasi akan otomatis menangkap:
    ⏰ Tanggal dan jam foto diambil
    📍 Lokasi GPS + alamat (jika device mendukung)
    Ini membantu mencegah pemalsuan bukti.

    📸 Tanpa foto pun boleh! Geolocation tetap tercatat otomatis setiap 30 detik.
  4. Klik + Tambah Pengeluaran

3. Kategori Pengeluaran & Budget Cap

Setiap kategori memiliki limit per hari yang berbeda tergantung jumlah engineer:

Kategori Cap Per Hari Keterangan
🚗 Transport Dalam Kota 75.000 × jumlah engineer Taksi, ojek, bus lokal
🍽️ Food & Beverage (Staff × 125.000) + (Manager × 200.000) Makanan dan minuman
🛣️ Transport ke Tujuan 120.000 × jumlah engineer Pesawat, kereta, mobil antar kota
🚆 Overhead Tidak ada limit Biaya tambahan lainnya
🎉 Entertain Tidak ada limit Hiburan, acara

4. Contoh Perhitungan Budget

Skenario 1: 2 Staff
• Transport Dalam Kota: 75.000 × 2 = 150.000/hari
• Food: 125.000 × 2 = 250.000/hari
• Transport ke Tujuan: 120.000 × 2 = 240.000/hari

Skenario 2: 1 Staff + 1 Manager
• Transport Dalam Kota: 75.000 × 2 = 150.000/hari
• Food: (125.000 × 1) + (200.000 × 1) = 325.000/hari
• Transport ke Tujuan: 120.000 × 2 = 240.000/hari

Skenario 3: 3 Manager
• Transport Dalam Kota: 75.000 × 3 = 225.000/hari
• Food: 200.000 × 3 = 600.000/hari
• Transport ke Tujuan: 120.000 × 3 = 360.000/hari

📑 Panduan Tab

Tab 1: Tambah (➕)

Tempat untuk:

  • Memilih trip yang aktif
  • Menambah pengeluaran baru
  • Melihat pengeluaran hari ini
  • Progress budget real-time

Tab 2: Ringkasan (📊)

Menampilkan:

  • Semua trip yang ada
  • Total budget, terpakai, dan sisa untuk setiap trip
  • Breakdown pengeluaran per kategori (grafik)
  • Persentase penggunaan budget

Tab 3: Rincian (📋)

Menampilkan:

  • Semua pengeluaran dari semua trip
  • Diurutkan berdasarkan tanggal (terbaru dulu)
  • Bisa lihat foto bukti dengan klik thumbnail
  • Bisa hapus pengeluaran

Tab 4: Export (📄)

Untuk:

  • Download laporan PDF per trip
  • Laporan mencakup tabel pengeluaran + semua foto bukti
  • Siap untuk di-submit ke pimpinan/akuntansi

📷 Fitur Foto Bukti & Geolocation

Cara Upload Foto

  1. Di form pengeluaran, cari section "Foto Bukti"
  2. Klik area dengan icon 📷
  3. Pilih dari:
    • Kamera - ambil foto langsung
    • Galeri - pilih dari foto yang sudah ada
  4. Saat upload foto, aplikasi akan meminta izin geolocation
    Izin Geolocation:
    • Klik "Allow" untuk mencatat lokasi
    • Klik "Block" jika tidak ingin (aplikasi tetap jalan)
  5. Preview foto akan muncul dengan info:
    • ⏰ Jam foto diambil
    • 📍 Koordinat GPS (jika izin diberikan)

Geolocation di Berbagai Device

Device Geolocation Timestamp
📱 iPhone/Android dengan GPS ✅ Koordinat GPS ✅ Selalu ada
🍎 iPad (WiFi only) ⚠️ "Tidak tersedia" ✅ Selalu ada
💻 Desktop/Laptop ⚠️ "Tidak tersedia" atau GPS eksternal ✅ Selalu ada
🌐 Browser tanpa izin ⚠️ "Izin ditolak" ✅ Selalu ada
💡 Mengapa Geolocation?
Geolocation dan timestamp membantu mencegah pemalsuan bukti pengeluaran dengan mencatat kapan dan di mana foto diambil.

📄 Export & Download PDF

Langkah-Langkah Export

  1. Buka tab Export
  2. Lihat daftar semua trip yang ada
  3. Klik tombol 📄 PDF pada trip yang ingin di-export
  4. Tunggu proses generate PDF (2-5 detik)
  5. File PDF otomatis ter-download

Isi PDF Laporan

Header: Nama trip, tujuan, nama karyawan, tanggal cetak

Summary: Total budget, terpakai, sisa, jumlah item

Tabel Pengeluaran: No, tanggal/jam, keterangan, kategori, jumlah

Bukti Foto: Foto lengkap + metadata (tanggal, jam, kategori, lokasi jika ada)

❓ FAQ (Pertanyaan Umum)

Bagaimana jika saya menambah pengeluaran melebihi budget category? ▼
Aplikasi akan menampilkan warning/alert berisi sisa budget yang masih available. Anda tidak bisa menambah pengeluaran jika melebihi limit. Hubungi atasan untuk approval budget tambahan terlebih dahulu.
Bisa kah saya mengedit pengeluaran yang sudah ditambahkan? ▼
Saat ini fitur edit belum tersedia. Untuk mengubah, Anda bisa hapus pengeluaran (klik ×) dan tambah ulang dengan data yang benar.
Apakah data saya aman? Dikirim ke server? ▼
Sangat aman. Semua data disimpan 100% lokal di device Anda menggunakan browser storage. Tidak ada yang dikirim ke server manapun. Sepenuhnya private dan offline-first.
Bagaimana kalau saya ganti device? Apakah data hilang? ▼
Data tersimpan lokal per device, jadi jika ganti device data akan hilang. Pastikan export PDF sebelum ganti device. Kami sedang mempertimbangkan fitur cloud sync di versi mendatang.
Berapa ukuran limit foto yang bisa di-upload? ▼
Tidak ada limit ukuran file, tapi semakin besar foto maka semakin banyak storage yang terpakai di browser. Rekomendasi: foto biasa (1-5 MB) untuk pengalaman optimal.
Berapa banyak trip yang bisa saya buat? ▼
Tidak ada limit formal, tapi tergantung kapasitas storage browser. Browser modern bisa handle ratusan trip dengan ribuan pengeluaran. Jika mendekati limit, export PDF dan bersihkan data lama.
Bagaimana cara hapus data/reset aplikasi? ▼
Buka developer tools (F12) → Console → ketik: localStorage.clear() → Enter → Refresh halaman. Data akan hilang semua, jadi pastikan sudah export PDF terlebih dahulu!

💡 Tips & Trik

✅ Gunakan kategori dengan benar
Kategorisasi yang tepat membuat laporan breakdown lebih akurat dan memudahkan verifikasi.
✅ Selalu upload foto bukti
Foto + timestamp + geolocation membuktikan pengeluaran authentic dan sulit dipalsukan.
✅ Update trip hari ini
Jangan menunggu akhir perjalanan. Update pengeluaran setiap hari agar tidak lupa.
✅ Cek progress bar harian
Monitor cap harian (transport & food) untuk menghindari overspending.
✅ Export PDF sebelum ganti device
Jika akan ganti device/clear cache, pastikan sudah download PDF laporan.
✅ Gunakan di mode offline
Aplikasi ini 100% offline. Anda bisa input data tanpa internet di lokasi manapun.

🔧 Troubleshooting

Geolocation tidak muncul di iPad/Desktop

Ini normal! iPad WiFi dan desktop umumnya tidak punya GPS. Timestamp tetap ter-record. Jika ingin GPS, gunakan smartphone dengan fitur location services aktif.

Foto tidak bisa di-upload

  • Pastikan browser support file input (semua modern browser support)
  • Cek format file (hanya gambar: JPG, PNG, WebP, dll)
  • Jika file terlalu besar, coba resize atau compress terlebih dahulu

PDF tidak ter-generate

  • Tunggu 2-5 detik, proses generate butuh waktu
  • Cek browser console (F12) untuk error message
  • Refresh halaman dan coba lagi

Data hilang setelah close browser

Kemungkinan besar:

  • Browser history dibersihkan (cache dan cookies terhapus)
  • Mode private/incognito (data tidak tersimpan di mode ini)
  • Storage penuh (clear storage browser untuk space lebih)

Solusi: Export PDF sebelum close browser untuk backup data.
